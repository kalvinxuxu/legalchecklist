## 测试工具

### 1. Locust 负载测试 (`performance_tests.py`)

**安装**: `pip install locust`

**运行命令**:
```bash
locust -f performance_tests.py --headless -u 100 -r 10 --run-time 3m --host http://localhost:8000
```

**测试场景**:
- `TestUser`: 模拟真实用户行为（浏览、登录、查看合同）
- `ApiLoadTest`: API 负载测试（高并发请求）

**任务权重**:
| 任务 | 权重 | 说明 |
|------|------|------|
| 首页访问 | 5 | 模拟用户访问首页 |
| 登录页访问 | 3 | 访问登录页面 |
| 工作区列表 | 10 | 获取工作区列表 |
| 合同列表 | 8 | 获取合同列表 |
| 合同详情 | 4 | 查看合同详情 |
| 审查结果 | 2 | 获取审查报告 |

---

### 2. API 基准测试 (`backend/scripts/benchmark.py`)

**运行命令**:
```bash
cd backend && python scripts/benchmark.py
```

**测试配置**:
- 并发用户数：1, 5, 10, 20, 50
- 每用户请求数：10
- 目标端点：登录、工作区列表、合同列表

**测试指标**:
- 平均响应时间
- 中位数响应时间
- P95 响应时间
- 标准差
- 吞吐量 (请求/秒)
- 错误率
