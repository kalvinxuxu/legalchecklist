## 详细需求

### 9.1-9.3: 合同理解辅助功能

在合同审查完成后，新增一个「理解」环节，使用 LLM 帮助用户快速了解合同：

**输出内容**:
1. **合同结构分析**
   - 识别合同的主要章节（如：前言、定义条款、保密义务、违约责任等）
   - 生成结构目录

2. **重要条款摘要**
   - 提取并摘要关键条款
   - 标记对用户的风险/利好点

3. **快速理解卡片**
   - 合同类型与用途
   - 关键日期/期限
   - 金额/责任上限
   - 双方核心义务

**技术实现**:
- 新增 `app/services/analysis/structure.py` - 结构分析服务
- 新增 `app/services/analysis/summary.py` - 摘要生成服务
- 更新审查流程：上传 → 审查 → **理解分析** → 展示结果

---

### 9.4-9.7: PDF 条款定位与高亮

**OCR 扫描**:
- 使用 `pdfplumber` 或 `pymupdf` 进行 PDF 文本提取
- 对于扫描版 PDF，集成 OCR 服务（如阿里云 OCR 或 Tesseract）
- 记录每段文本在 PDF 中的位置（页码、坐标）

**条款定位**:
- 根据审查结果中的风险条款原文
- 在 PDF 中检索匹配位置
- 记录页码和区域坐标

**高亮标注**:
- 生成带标注的 PDF 文件（使用 `pymupdf` 或 `reportlab`）
- 或用前端 PDF 渲染器实现高亮层（如 `pdf.js`）

**技术实现**:
- 新增 `app/services/pdf/highlighter.py` - PDF 高亮服务
- 新增 `app/services/pdf/locator.py` - 条款定位服务
- 新增 API: `GET /api/v1/contracts/{id}/highlighted-pdf` - 返回高亮 PDF

---

### 9.8-9.9: 前端展示

**合同理解卡片**:
- 在审查报告页面顶部展示
- 卡片包含：合同结构图、重要条款摘要、快速要点

**高亮 PDF 查看器**:
- 使用 `pdf.js` 渲染高亮 PDF
- 点击风险条款自动跳转到对应位置
- 支持缩放、翻页、高亮切换
