**日期**: 2026-03-31
**状态**: 项目已启动，准备执行 Phase 1

---

## 项目概述

打造面向中国中小企业的轻量化法务 AI 平台，专注于 NDA 和劳动合同的智能审查。

**价值主张**:
- 🎯 轻量化：开箱即用，注册即用
- 💰 可负担：¥299-2999/月
- 🔒 可信赖：RAG + 引用溯源
- 📱 多端覆盖：Web + 微信小程序

## 立即行动（Phase 1）

### Task 1.1: 购买域名并备案
- 访问阿里云官网
- 选择域名（建议：`hetong-ai.com` / `fatiao.ai`）
- 提交备案申请

### Task 1.2: 购买云服务器
- 访问腾讯云官网
- 选择轻量应用服务器（2 核 4G）
- 设置 SSH 密钥

### Task 1.3: 配置 Docker 环境
- 安装 Docker 和 Docker Compose
- 创建 `docker-compose.yml`
- 测试启动

### Task 1.4: 创建 Git 仓库
- 创建 GitHub/Gitee 仓库
- 初始化项目目录结构
- 首次提交

### Task 1.5: 申请智谱 AI API
- 访问 open.bigmodel.cn
- 注册账号
- 获取 API Key

### Task 1.6: 申请微信小程序
- 访问 mp.weixin.qq.com
- 注册小程序
- 获取 AppID
