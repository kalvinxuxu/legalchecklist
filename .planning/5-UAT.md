## 异步任务队列实现

### 开发环境
- 使用 `asyncio.create_task()` 后台处理
- 无需 Redis，简化开发配置

### 生产环境（可选）
- 使用 Celery + Redis
- 自动重试机制（最多 3 次）
- 超时保护（5 分钟软超时，10 分钟硬超时）
- 失败状态标记

### 配置方式
```env
# .env 文件
USE_CELERY=true
REDIS_URL=redis://localhost:6379/0
```

### 启动 Worker
```bash
python celery_worker.py
```
