## 项目概况

**项目名称**: 法务 AI SaaS 平台
**目标用户**: 中小企业（10-500 人规模）
**核心价值**: 让每家公司都能用得起专业的合同审查服务
**定价策略**: ¥299-2999/月，仅为传统法律服务的 1/10

## 技术选型（国内版）

| 层级 | 选型 |
|------|------|
| 前端 | Vue 3 + Vite + Element Plus |
| 小程序 | uni-app（微信/支付宝/抖音） |
| 后端 | FastAPI (Python) |
| 数据库 | MySQL 8.0 (阿里云 RDS) |
| 向量检索 | Elasticsearch 8.x |
| LLM | 智谱 GLM-4 |
| Embedding | 智谱 embedding-2 |
| 部署 | Docker Compose + 云服务器 |

## MVP 范围

**Phase 1 核心功能**（第 1-4 周）:
- 用户系统（邮箱/手机 + 微信登录）
- 租户管理（多租户隔离）
- 合同上传（PDF/Word）
- NDA 审查
- 劳动合同审查
- 审查报告（风险标记 + 修改建议 + 法条引用）

## 关键决策

1. **多租户架构**: 共享数据库 + 行级隔离（成本低，适合 SMB）
2. **RAG 架构**: 公共知识库 + 租户私有库混合检索
3. **LLM 选择**: 智谱 GLM-4（法律场景表现好，国内可访问）
4. **发布渠道**: Web + 微信小程序双端
