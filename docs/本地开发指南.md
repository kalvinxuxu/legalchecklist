# 本地开发环境配置指南（无需 Docker）

## 前提条件

- Python 3.11+
- Node.js 18+
- DeepSeek API Key

---

## 快速开始

### 步骤 1: 配置环境变量

进入后端目录，复制并编辑 `.env` 文件：

```bash
cd backend
cp .env.example .env
```

编辑 `.env` 文件，填入你的 DeepSeek API Key：

```bash
# 使用 SQLite（推荐本地开发）
DATABASE_TYPE=sqlite
SQLITE_PATH=./legal_saas.db

# DeepSeek API
DEEPSEEK_API_KEY=sk-your-api-key-here

# JWT
JWT_SECRET=dev-secret-key-123456
```

### 步骤 2: 启动后端服务

**Windows**:
```bash
cd backend
start-dev.bat
```

**Mac/Linux**:
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python scripts/init_db.py
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

启动后访问：http://localhost:8000/docs

### 步骤 3: 启动前端服务

```bash
cd frontend
npm install
npm run dev
```

启动后访问：http://localhost:5173

## 数据库说明

### SQLite（本地开发）

无需安装，自动创建数据库文件：

```bash
# 数据库文件位置
backend/legal_saas.db
```

### MySQL（生产环境）

修改 `.env` 配置：

```bash
DATABASE_TYPE=mysql
DATABASE_URL=mysql://root:password@localhost:3306/legal_saas
```

## 常见问题

### Q: Python 版本不兼容

确保使用 Python 3.11：
```bash
python --version
```

### Q: 依赖安装失败

尝试使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 前端页面空白

检查控制台是否有报错，确保后端服务已启动。

### Q: 401 Unauthorized

Token 可能过期或未携带，检查请求头：
```javascript
Authorization: Bearer <your-token>
```
