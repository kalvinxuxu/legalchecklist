# 法务 AI SaaS - 服务器部署指南

## 前提条件

- 已购买云服务器（推荐腾讯云轻量应用服务器，2 核 4G 以上）
- 系统：Ubuntu 20.04 或 22.04
- 已有 DeepSeek API Key

---

## 方式一：一键部署脚本（推荐）

### 步骤 1: 上传项目代码到服务器

**选项 A - 使用 Git**:
```bash
# SSH 登录服务器
ssh root@你的服务器 IP

# 克隆项目
git clone <你的仓库地址> /opt/legal-ai-saas
cd /opt/legal-ai-saas
```

**选项 B - 使用 SCP（从本地上传）**:
```bash
# 在本地 PowerShell 执行
scp -r ./* root@你的服务器 IP:/opt/legal-ai-saas
```

### 步骤 2: 运行部署脚本

```bash
# 进入项目目录
cd /opt/legal-ai-saas

# 执行部署脚本
chmod +x scripts/deploy.sh
bash scripts/deploy.sh
```

### 步骤 3: 配置环境变量

脚本会提示你配置 `.env` 文件：

```bash
cd /opt/legal-ai-saas/backend
cp .env.example .env
vi .env
```

**必须配置的变量**:
```bash
DATABASE_TYPE=mysql
DATABASE_URL=mysql://root:你的密码@db:3306/legal_saas
DEEPSEEK_API_KEY=sk-你的 API Key
JWT_SECRET=随机生成的密钥
```

### 步骤 4: 启动服务

```bash
cd /opt/legal-ai-saas/docker
docker compose up -d
```

## 方式二：手动部署

### 1. 安装 Docker

```bash
# 更新包索引
apt update

# 安装 Docker
curl -fsSL https://get.docker.com | bash

# 启动 Docker
systemctl enable docker
systemctl start docker

# 安装 Docker Compose
apt install docker-compose-plugin
```

### 2. 配置项目

```bash
# 创建项目目录
mkdir -p /opt/legal-ai-saas
cd /opt/legal-ai-saas

# 上传或克隆代码
# ...（见上方）

# 配置环境变量
cd backend
cp .env.example .env
vi .env
```

### 3. 配置 docker-compose.yml

编辑 `docker/docker-compose.yml`，确保环境变量正确：

```yaml
services:
  app:
    environment:
      - DATABASE_URL=mysql://root:你的密码@db:3306/legal_saas
      - DEEPSEEK_API_KEY=你的 API Key
      - JWT_SECRET=随机密钥
```

### 4. 启动服务

```bash
cd /opt/legal-ai-saas/docker
docker compose up -d
```

### 5. 验证服务

```bash
# 查看服务状态
docker compose ps

# 查看应用日志
docker compose logs -f app

# 测试 API
curl http://localhost/health
```

## 配置 Nginx（可选，用于生产环境）

### 1. 安装 Nginx

```bash
apt install nginx -y
```

### 2. 配置反向代理

创建 `/etc/nginx/sites-available/legal-saas`:

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:80;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 3. 启用配置

```bash
ln -s /etc/nginx/sites-available/legal-saas /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

## 配置 HTTPS（推荐）

使用 Let's Encrypt 免费证书：

```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 获取证书
certbot --nginx -d your-domain.com
```

## 常用运维命令

```bash
# 查看服务状态
docker compose ps

# 查看日志
docker compose logs -f app
docker compose logs -f db

# 重启服务
docker compose restart

# 停止服务
docker compose down

# 更新代码并重启
cd /opt/legal-ai-saas
git pull
docker compose up -d

# 进入容器
docker compose exec app bash
docker compose exec db bash

# 备份数据库
docker compose exec db mysqldump -uroot -p legal_saas > backup.sql
```

## 故障排查

### 容器启动失败

```bash
# 查看详细日志
docker compose logs app

# 检查配置
docker compose config
```

### 数据库连接失败

```bash
# 检查 MySQL 是否启动
docker compose ps db

# 测试连接
docker compose exec db mysql -uroot -p -e "SHOW DATABASES;"
```

### API 无法访问

```bash
# 检查防火墙
ufw status

# 检查端口监听
netstat -tlnp | grep :80

# 检查 Nginx 配置
nginx -t
```
