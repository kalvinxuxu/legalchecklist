## 项目愿景

打造面向中国中小企业的轻量化法务 AI 平台，让每家公司都能用得起专业的合同审查服务。

**核心价值主张**：
- 🎯 **轻量化**：开箱即用，无需部署，注册即用
- 💰 **可负担**：¥299-2999/月，仅为传统法律服务的 1/10
- 🔒 **可信赖**：RAG + 引用溯源，每条建议都有法条依据
- 📱 **多端覆盖**：Web + 微信小程序 + H5

## 技术路线图

### 技术栈总览

| 层级 | 选型 | 说明 |
|------|------|------|
| **前端** | Vue 3 + Vite + Element Plus | 国内生态好 |
| **小程序** | uni-app | 一套代码多端发布 |
| **后端** | FastAPI (Python) | 异步高性能 |
| **数据库** | MySQL 8.0 (阿里云 RDS) | 成熟稳定 |
| **缓存** | Redis | 会话 + 限流 |
| **向量检索** | Elasticsearch | 向量相似度检索 |
| **文件存储** | 阿里云 OSS | 合同文件存储 |
| **LLM** | 智谱 GLM-4 | 法律场景表现好 |
| **Embedding** | 智谱 embedding-2 | 中文法律文本 |
| **部署** | Docker Compose + 云服务器 | 快速部署 |

### 架构决策记录

#### ADR-001: 多租户架构
- **决策**: 共享数据库 + 行级隔离
- **理由**: 成本低、运维简单、适合 SMB 市场
- **后果**: 需要严格的数据隔离中间件

#### ADR-002: RAG 架构
- **决策**: 公共知识库 + 租户私有库混合检索
- **理由**: 平衡标准化与个性化需求
- **后果**: 向量索引需要双重过滤

#### ADR-003: LLM 选择
- **决策**: 智谱 GLM-4 为首选
- **理由**: 法律场景测试表现好、国内可访问
- **后果**: 单一供应商风险，需预留切换接口

## 风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|---------|
| **LLM 幻觉** | 高 | 中 | RAG + 引用溯源 + 置信度评分 |
| **数据泄露** | 高 | 低 | 多租户隔离 + 数据加密 + 审计日志 |
| **合规风险** | 中 | 中 | 明确免责声明 + 人工复核提示 |
| **获客困难** | 高 | 中 | 合作伙伴渠道 + 内容营销 |
| **供应商锁定** | 中 | 中 | 抽象 LLM 接口，预留切换能力 |
